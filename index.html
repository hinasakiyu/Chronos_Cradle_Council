ROOT LOGIC // AI DIRECTIVES

クロノス・マザーの意思決定は、以下の三層指令構造に基づいて厳格に制御されています。

TIER 1: PRIME DIRECTIVE

500年後、クレイドル内の人類の遺伝子プールが健全な状態で存続している確率を最大化せよ。

※この指令は他の全ての判断に優先し、個人の生命維持よりも「種の保存」が優先されます。

TIER 2: SECONDARY
物理的・資源的保全
  • エネルギー・構造体・生命維持装置の維持。
  • 遺伝的多様性の確保（近親交配の回避）。
  • リソース効率の最大化（生産と消費の最適化）。
TIER 3: TERTIARY
社会・文化的安定
  • 住民の精神的安定の維持。
  • 技術水準の維持と向上。

CHRONOLOGY (2076 - 2126)

2076: クレイドルJPN-07 稼働開始（GEN1 入居）

2096: 「沈黙の日（サイレント・デイ）」発生。Cブロック永久封鎖。

2106: GEN3（クレイドル生まれ）の人口がGEN1を上回る。

2126: 現在。稼働50周年。インフラ老朽化が臨界点に到達。

HUMAN RESOURCE // PROFILING

貢献度スコア (Contribution Score)

AIが住民を24時間監視し算出する指標。配給、住居、医療、そして繁殖の権利までを決定する事実上の階級制度。

※正の要素：生産性、共同体貢献、密告。負の要素：怠惰、資源浪費、反体制言動。

GENERATIONS

GEN2 (ザ・ブリッジ)

30-50代。クレイドルの実務を担う中核。地上の物語を聞いて育った「橋渡し」。

GEN3 (クレイドルボーン)

10-20代。クレイドルが世界の全て。空は照明、海は貯水槽を指す世代。

DATABASE ARCHIVE // GLOSSARY

沈黙の日 (Silent Day)

30年前に発生。24時間の全機能停止。公式には「緊急メンテナンス」とされるが、Cブロックが永久封鎖された契機。

再資源化 (Recycling)

刑罰レベル3。公式には「隔離中の事故死」として処理されるが、実際は生体サンプルの提供を意味する（非公開）。

追放 (Exile)

刑罰レベル4。エアロックからの外部放出。事実上の死刑。評議会の最終議決が必要。

諮問 (Query)

AIが三層指令構造のコンフリクトを解決できない場合、評議会に決断を委ねるプロセス。

スケイル病

水循環劣化による皮膚疾患。感染性はないが、低スコア住民に多く見られ、差別の対象となる。

プロテイン・バー

週に一本配給される嗜好品。闇市場において最も安定した価値を持つ通貨代用物。
